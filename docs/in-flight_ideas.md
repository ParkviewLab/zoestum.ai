## Optimize the logo

`assets/img/zoestum_logo.svg` is a ~169 KB Inkscape/Gemini export carrying editor
metadata (`sodipodi:`/`inkscape:` attributes, `<namedview>`) and high path
precision. Run it through `svgo` (strip that cruft, round coordinates) to cut the
file size substantially before the site grows past coming-soon. The current logo is
also a *rough* draft — replace it when a final mark is ready.
